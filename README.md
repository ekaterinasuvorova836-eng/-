Как запустить проект
Установите Python (версии 3.7 или новее), если он ещё не установлен.
Сохраните код. Скопируйте содержимое book_tracker.py в файл с таким именем.
Создайте файлы. В той же папке создайте пустой файл books.json и файл .gitignore с указанным содержимым.
Запустите программу. Откройте терминал/командную строку в этой папке и выполните команду:
bash
Копировать
python book_tracker.py
Используйте меню. Для сохранения и загрузки данных используйте меню «Файл» в верхней части окна приложения.
Инициализируйте Git.
bash
Копировать
git init
git add .
git commit -m "Initial commit"
git remote add origin <ссылка_на_ваш_репозиторий_на_GitHub>
